Machine learning snake game


Welcome to "Quiz Snake: Machine Learning Challenge"!

This game mixes the classic Snake game with fun questions about machine learning.

Each round starts with a question and four answer options, each linked to a colored block (Red, Green, Blue, Yellow). Control the snake to eat the block with the correct answer. If you're right, the snake grows longer. If you're wrong, it shrinks.

New questions come after each answer, but avoid crashing into the walls or yourself, or the game ends. Test your knowledge and see how long you can grow the snake!
